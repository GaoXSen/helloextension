Hello Extension (Chrome Extension)

Features
- Popup with quick actions: Globe, Galaxy, Calculator, Snake, 3D Snake
- Globe: Canvas 2D rotating Earth with stars and glow
- Galaxy: Spiral galaxy with starfield and core glow
- Calculator: Scientific calculator with DEG/RAD, memory, functions
- Snake: Neon-styled 2D snake with multiple skins and mobile controls
- 3D Snake: WebGL board with rounded snake segments and bright boundaries

Install (Chrome / Edge)
- Open `chrome://extensions` (or `edge://extensions`)
- Enable Developer mode
- Click "Load unpacked" and select this folder

Dev Notes
- No external libraries; all pages are static HTML/JS/CSS
- Entry point: `manifest.json` (MV3)
